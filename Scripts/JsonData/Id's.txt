Id Listing

Each number is the entity's UID.
1-1000 - NPC Entities.
1001-2000 - Item Entities.
2001-3000 - Terrain Entities.

Entity Listing:

0: Player Character
1: Blacklog Spider
2: Deepwater Eel
3: Boring Beetle
1001: Broken Blade
1100: Potion of Explosion
1300: Scroll of Lightning
2001: Door
2002: Chest
2300: Web
2500: Variable Statue (These are statues created from creatures turning to stone, this ID is used to represent them as a whole.)
